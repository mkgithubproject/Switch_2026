# ЁЯза What is `10.0.0.0/16`?

ЁЯСЙ ItтАЩs a **CIDR block** used to define a network range in a VPC.

# ЁЯзй What does `/16` mean?

Use this formula:

\text{Host bits} = 32 - 16 = 16 \quad\Rightarrow\quad \text{Total IPs} = 2^{16} = 65536

ЁЯСЙ So:

* Total IPs = **65,536**

# ЁЯза Step 1: IP in Binary

Take:

```text
10.0.0.0
```

Convert each octet (8 bits each):

```text
10  = 00001010
0   = 00000000
0   = 00000000
0   = 00000000
```

ЁЯСЙ Full 32-bit IP:

```text

```

---

# ЁЯзй Step 2: What `/16` Means

ЁЯСЙ `/16` = first **16 bits are fixed**, rest **16 bits are free**

```text
[ Network (fixed) ] [ Host (free) ]

00001010.00000000 | 00000000.00000000
   (10)     (0)        (X)      (X)
```

---

# ЁЯФе Step 3: Find the LAST IP

To get the **maximum range**, set all free bits = 1

```text

```

---

# ЁЯза Step 4: Convert back to Decimal

Now convert `11111111`:

11111111_2 = 2^8 - 1 = 255

ЁЯСЙ So:

```text
11111111 = 255
```
