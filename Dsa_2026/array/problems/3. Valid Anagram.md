# Refined Problem Statement

You are given two strings `s` and `t`.

Determine whether `t` is an anagram of `s`.

Two strings are considered anagrams if:

* they contain the exact same characters
* each character appears the same number of times
* the order of characters does not matter

Return:

* `true` if `t` is an anagram of `s`
* `false` otherwise

# Approaches for Valid Anagram

Problem:
Given two strings `s` and `t`, return `true` if `t` is an anagram of `s`, otherwise return `false`.

---

# Approach 1 — Brute Force

## Idea

For every character in `s`:

* search for the same character in `t`
* remove matched character from `t`
* if any character not found → not an anagram

---

## Code

```js id="r7p2kx"
function isAnagram(s, t) {
    if (s.length !== t.length) {
        return false;
    }

    t = t.split('');

    for (let char of s) {
        let index = t.indexOf(char);

        if (index === -1) {
            return false;
        }

        t.splice(index, 1);
    }

    return true;
}
```

# Approach 2 — Sorting

## Idea

If two strings are anagrams:

* sorting both strings will make them equal

---

## Code

```js id="m8t3ql"
function isAnagram(s, t) {
    if (s.length !== t.length) {
        return false;
    }

    let sortedS = s.split('').sort().join('');
    let sortedT = t.split('').sort().join('');

    return sortedS === sortedT;
}
```

# Approach 3 — Hash Map / Frequency Counter (Best)

## Idea

Count frequency of characters.

If both strings have same frequencies:
→ anagram

---

## Code

```js id="q9v5ta"
function isAnagram(s, t) {
    if (s.length !== t.length) {
        return false;
    }

    let freq = {};

    // Count characters of s
    for (let char of s) {
        freq[char] = (freq[char] || 0) + 1;
    }

    // Decrease count using t
    for (let char of t) {
        if (!freq[char]) {
            return false;
        }

        freq[char]--;
    }

    return true;
}
```

# Approach 4 — Array Frequency (Most Optimized)

Since only lowercase English letters exist,
use an array of size `26`.

---

## Code

```js id="j4t9mw"
function isAnagram(s, t) {
    if (s.length !== t.length) {
        return false;
    }

    let count = new Array(26).fill(0);

    for (let i = 0; i < s.length; i++) {
        count[s.charCodeAt(i) - 97]++;
        count[t.charCodeAt(i) - 97]--;
    }

    for (let num of count) {
        if (num !== 0) {
            return false;
        }
    }

    return true;
}
```

---

# Why `97`?

ASCII value of:

```js id="p3n7la"
'a' = 97
```

So:

```js id="y1v8qx"
charCodeAt() - 97
```

converts:

* `a → 0`
* `b → 1`
* `c → 2`
