# Problem Statement

You are given an array of strings `strs`.

Your task is to find the **longest common prefix** shared by all strings in the array.

A **prefix** is a group of characters that appears at the beginning of a string.

Return:

* the longest common starting substring among all strings
* or an empty string `""` if no common prefix exists

---

# Example 1

Input:

```js id="q7m2vk"
strs = ["flower","flow","flight"]
```

Explanation:

Common starting characters:

```js id="p4x8ta"
"fl"
```

Because:

* `"flower"` starts with `"fl"`
* `"flow"` starts with `"fl"`
* `"flight"` starts with `"fl"`

But:

```js id="r1v6wc"
"flo"
```

is not common to all strings.

Output:

```js id="m9q3kp"
"fl"
```

---

# Example 2

Input:

```js id="y5t1rn"
strs = ["dog","racecar","car"]
```

Explanation:

The strings do not share any starting character.

Output:

```js id="c8v4px"
""
```

(empty string)
