# Problem Statement

You are given an array of strings `strs`.

Your task is to find the **longest common prefix** shared by all strings in the array.

A **prefix** is a group of characters that appears at the beginning of a string.

Return:

* the longest common starting substring among all strings
* or an empty string `""` if no common prefix exists

---

# Example 1

Input:

```js id="q7m2vk"
strs = ["flower","flow","flight"]
```

Explanation:

Common starting characters:

```js id="p4x8ta"

```

Because:

* `"flower"` starts with `"fl"`
* `"flow"` starts with `"fl"`
* `"flight"` starts with `"fl"`

But:

```js id="r1v6wc"

```

is not common to all strings.

Output:

```js id="m9q3kp"
"fl"
```

# Approaches for Longest Common Prefix

Problem:
Find the longest starting substring common to all strings.

---

# Approach 1 — Horizontal Scanning (Best Beginner Approach)

## Idea

Take the first string as prefix.

Compare it with every other string:

* if mismatch occurs
* keep shortening prefix
* until all strings match

---

# Code

```js id="p7m2kx"
function longestCommonPrefix(strs) {

    let prefix = strs[0];

    for (let i = 1; i < strs.length; i++) {

        while (!strs[i].startsWith(prefix)) {
            prefix = prefix.slice(0, prefix.length - 1);
        }

    }

    return prefix;
}
```

# Approach 2 — Vertical Scanning

## Idea

Compare characters column by column.

Example:

```js id="d1q9wp"
flower
flow
flight
```

Check:

* first character of all strings
* second character of all strings
* continue until mismatch

---

# Code

```js id="m8v4kc"
function longestCommonPrefix(strs) {

    for (let i = 0; i < strs[0].length; i++) {

        let char = strs[0][i];

        for (let j = 1; j < strs.length; j++) {

            if (
                i >= strs[j].length ||
                strs[j][i] !== char
            ) {
                return strs[0].slice(0, i);
            }

        }
    }

    return strs[0];
}
```

# Approach 3 — Sorting Trick

## Idea

After sorting:

* most different strings become first and last
* common prefix of those two is answer

---

# Why?

Example:

```js id="w9p2rn"
["flower","flow","flight"]
```

Sorted:

```js id="q6m1xa"
["flight","flow","flower"]
```

Now compare:

```js id="z3v7tk"
first = "flight"
last = "flower"
```

Common prefix:

```js id="p8q4wc"
"fl"
```

---

# Code

```js id="g5m9ra"
function longestCommonPrefix(strs) {

    strs.sort();

    let first = strs[0];
    let last = strs[strs.length - 1];

    let i = 0;

    while (
        i < first.length &&
        first[i] === last[i]
    ) {
        i++;
    }

    return first.slice(0, i);
}
```

# Approach 4 — Trie (Advanced)

## Idea

Insert all strings into Trie.

Traverse until:

* node has more than one child
* or word ends

Collected path becomes prefix.
