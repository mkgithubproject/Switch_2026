# Refined Problem Statement

You are given:

* an integer array `nums`
* an integer `target`

Your task is to find **two different elements** in the array whose sum equals `target`.

Return the indices of those two elements.

---

# Important Conditions

* Each input has **exactly one valid solution**
* You cannot use the same element twice
* The returned indices can be in any order

---

# Example 1

Input:

```js id="u4m8kp"
nums = [2,7,11,15]
target = 9
```

Explanation:

```js id="q7t2vc"
nums[0] + nums[1]
= 2 + 7
= 9
```

So the answer is:

```js id="d1x5ra"
[0,1]
```

---

# In Simple Words

Find:

```js id="h2k7zb"
nums[i] + nums[j] = target
```

where:

```js id="n6p1vc"
i != j
```

Then return:

```js id="a4x8qt"
[i, j]
```
