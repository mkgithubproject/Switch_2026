# 2. Prefix Sum

This is one of the MOST IMPORTANT interview patterns.

Companies like Google and Amazon ask this pattern again and again.

Why?

Because Prefix Sum helps optimize problems from:

```text id="nm50f2"
O(n²) → O(n)
```

---

# What is Prefix Sum?

Prefix Sum means:

> Store cumulative sum from beginning up to every index.

---

# Real Life Analogy (Never Forget)

Imagine your bank balance history.

Daily earnings:

```text id="c33h9e"
Day 1 → 100
Day 2 → 200
Day 3 → 50
Day 4 → 300
```

Now someone asks:

```text id="f2zybi"
How much money earned till Day 3?
```

Instead of recalculating every time:

```text id="p8n9li"
100 + 200 + 50
```

You pre-store totals.

That stored running total is:

```text id="j8vq4q"
PREFIX SUM
```

# Basic Idea

Given:

```text id="7m0p03"
arr = [2,4,6,8]
```

Prefix Sum Array becomes:

```text id="xk8bsk"
[2,6,12,20]
```

Because:

```text id="o5qepc"
2
2+4 = 6
2+4+6 = 12
2+4+6+8 = 20
```

# Building Prefix Sum Array

Code:

```javascript id="7d8db4"
const arr = [2, 4, 6, 8];

const prefix = [];

prefix[0] = arr[0];

for (let i = 1; i < arr.length; i++) {
    prefix[i] = prefix[i - 1] + arr[i];
}

console.log(prefix);
```

Output:

```text id="5nj4it"
[2, 6, 12, 20]
```

---

# Dry Run

Initial:

```text id="d9a1yj"
prefix[0] = 2
```

Step 1:

```text id="hy3v06"

          = 2 + 4

```

Step 2:

```text id="e41zmx"

```

Step 3:


# MOST IMPORTANT USE CASE

# Range Sum Query

This is THE REAL reason Prefix Sum exists.

---

# Problem

Given:

```text id="mhn25v"
arr = [2,4,6,8,10]
```

Find sum from index:

```text id="j0hz8z"
1 to 3
```

Meaning:

```text id="v2lv3h"
4 + 6 + 8 = 18
```

---

# Brute Force Approach

```javascript id="6udw38"
let sum = 0;

for (let i = left; i <= right; i++) {
    sum += arr[i];
}
```

Time Complexity:

```text id="50nv5w"
O(n)
```

If many queries come:

```text id="8o71r2"
Very slow
```

---

# Prefix Sum Magic

Using prefix array:

```text id="m2k66s"
[2,6,12,20,30]
```

Formula:

If left = 0

```text id="i4cjq7"
prefix[right]
```

Else:

```text id="cl68z9"
prefix[right] - prefix[left - 1]
```

---

# Why It Works

Need:

```text id="0a4mfi"
4 + 6 + 8
```

Prefix till 3:

```text id="zllwnp"
20
```

Prefix till before left:

```text id="wzhx1e"
2
```

Subtract:

```text id="kwlkrn"
20 - 2 = 18
```

---

# Formula (SUPER IMPORTANT)

\text{Range Sum} = \text{prefix}[r] - \text{prefix}[l-1]

---

# Complete Code

```javascript id="t2k80z"
const arr = [2, 4, 6, 8, 10];

const prefix = [];

prefix[0] = arr[0];

for (let i = 1; i < arr.length; i++) {
    prefix[i] = prefix[i - 1] + arr[i];
}

const left = 1;
const right = 3;

let result;

if (left === 0) {
    result = prefix[right];
} else {
    result = prefix[right] - prefix[left - 1];
}

console.log(result);
```

Output:

```text id="k7d2rj"
18
```
