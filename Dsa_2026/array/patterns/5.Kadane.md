# Core Decision at Every Index

At every element:

Kadane asks:

```text id="k19"
Should I:

1. continue previous subarray
OR
2. start fresh from current element
```

# CODE (VERY IMPORTANT)

```javascript id="k48"
function kadane(arr) {

    let currentSum = arr[0];
    let maxSum = arr[0];

    for (let i = 1; i < arr.length; i++) {

        currentSum = Math.max(arr[i], currentSum + arr[i]);

        maxSum = Math.max(maxSum, currentSum);
    }

    return maxSum;
}

console.log(kadane([-2,1,-3,4,-1,2,1,-5,4]));
```

# MOST IMPORTANT INTERVIEW INSIGHT

Kadane is actually:

# Dynamic Programming

Because:

At index i:

```text id="k50"

```

Specifically:

```text id="k51"
best subarray ending at i
```

depends on:

```text id="k52"
best subarray ending at i-1
```

# DP Relation

dp[i] = \max(arr[i],\ dp[i-1] + arr[i])

Kadane is optimized DP.

# Tiny Challenge

Solve using Kadane:

```text id="k64"
[5,-2,3,4]
```

Expected answer:

```text id="k65"
10
```

Subarray:

```text id="k66"
[5,-2,3,4]
```

```
function kadane(arr) {

    let currentSum = arr[0];
    let maxSum = arr[0];

    for (let i = 1; i < arr.length; i++) {

        // Decide:
        // continue previous subarray
        // OR start new from current element

        if (currentSum + arr[i] > arr[i]) {
            currentSum = currentSum + arr[i];
        } else {
            currentSum = arr[i];
        }

        // update maxSum manually

        if (currentSum > maxSum) {
            maxSum = currentSum;
        }
    }

    return maxSum;
}

console.log(kadane([-2,1,-3,4,-1,2,1,-5,4]));
```
