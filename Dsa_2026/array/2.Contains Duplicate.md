# Problem Statement

You are given an integer array `nums`.

Your task is to check whether the array contains any duplicate element.

* Return `true` if any number appears more than once.
* Return `false` if all elements in the array are unique.

---

# Example 1

Input:

```js id="f7k2mz"
nums = [1,2,3,1]
```

Explanation:

* Number `1` appears two times.

Output:

```js id="r3p8vd"
true
```

---

# Example 2

Input:

```js id="y5t1qa"
nums = [1,2,3,4]
```

Explanation:

* Every element appears only once.

Output:

```js id="m9w6cn"
false
```

---

# Example 3

Input:

```js id="b4e7lu"
nums = [1,1,1,3,3,4,3,2,4,2]
```

Explanation:

* Multiple elements are repeated (`1`, `3`, `4`, `2`).

Output:

```js id="p2z8xh"
true
```

---

# In Simple Words

Check whether:

* any element is repeated → return `true`
* all elements are different → return `false`
