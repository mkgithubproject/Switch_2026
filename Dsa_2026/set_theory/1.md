# Set in JavaScript

A **Set** is a built-in JavaScript object that stores **unique values only**.

That means:

* duplicate values are automatically removed
* each value can exist only once

---

# Creating a Set

```js id="a7d2kp"
let set = new Set();
```

Or with values:

```js id="m4q8rv"
let set = new Set([1, 2, 3]);
```

---

# Important Property of Set

```js id="v9n5tx"
let set = new Set([1, 2, 2, 3]);

console.log(set);
```

Output:

```js id="r6c1jw"
Set(3) {1, 2, 3}
```

Duplicate `2` is removed automatically.

---

# Common Set Methods

---

## 1. add()

Adds a value.

```js id="u2f7md"
let set = new Set();

set.add(10);
set.add(20);

console.log(set);
```

Output:

```js id="x5q3la"
Set(2) {10, 20}
```

---

## 2. has()

Checks whether value exists.

```js id="z8p1ks"
console.log(set.has(10)); // true
console.log(set.has(50)); // false
```

---

## 3. delete()

Removes a value.

```js id="h3v9qe"
set.delete(10);

console.log(set);
```

---

## 4. size

Returns number of elements.

```js id="n6r2wc"
console.log(set.size);
```

---

## 5. clear()

Removes all elements.

```js id="e1m7tb"
set.clear();
```

---

# Looping Through a Set

```js id="g4k8yu"
let set = new Set([1,2,3]);

for (let value of set) {
    console.log(value);
}
```

---

# Convert Array → Set

```js id="p9t5dx"
let arr = [1,2,2,3];

let set = new Set(arr);

console.log(set);
```

---

# Convert Set → Array

Using spread operator:

```js id="w2c6mn"
let set = new Set([1,2,3]);

let arr = [...set];

console.log(arr);
```

---

# Real Interview Usage

---

## Remove Duplicates from Array

```js id="q7b1fr"
let nums = [1,2,2,3,3,4];

let unique = [...new Set(nums)];

console.log(unique);
```

Output:

```js id="l5x8zp"
[1,2,3,4]
```

---

## Contains Duplicate Problem

```js id="f3n7ua"
function containsDuplicate(nums) {
    return new Set(nums).size !== nums.length;
}
```

# Time Complexity

| Operation | Complexity |
| --------- | ---------- |
| add()     | O(1)       |
| has()     | O(1)       |
| delete()  | O(1)       |

Very fast because Set uses hashing internally.

---

# Set vs Array

| Feature           | Array  | Set    |
| ----------------- | ------ | ------ |
| Duplicate Allowed | ✅ Yes  | ❌ No   |
| Ordered           | ✅ Yes  | ✅ Yes  |
| Fast Search       | ❌ O(n) | ✅ O(1) |
| Index Access      | ✅ Yes  | ❌ No   |

---

# Important Note

Set stores:

* numbers
* strings
* objects
* arrays
* anything

But objects are compared by reference:

```js id="y4p2kc"
let set = new Set();

set.add({name: "Mohit"});
set.add({name: "Mohit"});

console.log(set.size);
```

Output:

```js id="d8m6vq"
2
```

Because both objects are different references in memory.
