# DSA + Algorithms Roadmap for Product-Based Companies

If you're targeting companies like Google, Amazon, Microsoft, Flipkart, Uber, Swiggy, Zomato, or strong startups, then your DSA preparation should focus on:

* Problem-solving ability
* Pattern recognition
* Time complexity optimization
* Clean coding
* Communication during interviews

Since you already know backend development and Java, this roadmap is optimized for **interview-focused preparation**.

---

# Phase 1 — DSA Fundamentals (Very Important)

## 1. Time Complexity & Space Complexity

Learn:

* Big O notation
* Best / Average / Worst case
* Common complexities:

  * O(1)
  * O(log n)
  * O(n)
  * O(n log n)
  * O(n²)

### Must Know:

* How loops affect complexity
* Nested loops
* Recursion complexity
* HashMap average complexity

# Most Important Advice

Don’t randomly solve problems.

Instead:

1. Learn one pattern
2. Solve 5–10 similar questions
3. Understand optimization
4. Revise later

That builds interview intuition much faster.
