Learning React from basic to advanced works best when you build things progressively instead of memorizing APIs.

Here’s a roadmap that takes you from “never used React” to building production-level apps.

---

# Phase 1 — Understand the Foundation

Before React, you should know basic JavaScript well.

## Required JavaScript Topics

Learn these first:

* Variables (`let`, `const`)
* Functions
* Arrow functions
* Arrays & objects
* Array methods:

  * `map`
  * `filter`
  * `find`
* Destructuring
* Spread operator (`...`)
* Promises
* `async/await`
* ES modules (`import/export`)
* DOM basics
* Events

Example:

```js
const users = [
  { name: "Mohit", age: 25 },
  { name: "John", age: 30 }
];

const names = users.map(user => user.name);

console.log(names);
```

Output:

```js
["Mohit", "John"]
```

React uses this style heavily.

# Phase 2 — What is React?

React is a JavaScript library for building UI.

Instead of manually changing HTML:

```js
document.getElementById("title").innerText = "Hello";
```

React updates UI automatically based on state.

---

# Phase 3 — Setup React Project

Use Vite (modern & fast).

## Create App

```bash
npm create vite@latest my-react-app
```

Choose:

* Framework → React
* Variant → TypeScript or JavaScript

Then:

```bash
cd my-react-app
npm install
npm run dev
```

# Phase 4 — Understand JSX

JSX lets you write HTML inside JavaScript.

Example:

```jsx
function App() {
  return <h1>Hello React</h1>;
}
```

React converts this into JavaScript internally.

---

# Phase 5 — Components

Components are reusable UI blocks.

## Example

```jsx
function Welcome() {
  return <h1>Welcome User</h1>;
}

export default Welcome;
```

Use component:

```jsx
<Welcome />
```

---

# Phase 6 — Props

Props pass data to components.

## Example

```jsx
function User(props) {
  return <h1>Hello {props.name}</h1>;
}
```

Usage:

```jsx
<User name="Mohit" />
<User name="John" />
```

Better version:

```jsx
function User({ name }) {
  return <h1>Hello {name}</h1>;
}
```

---

# Phase 7 — State (`useState`)

State stores changing data.

## Example Counter

```jsx
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <h1>{count}</h1>

      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
    </div>
  );
}
```

# Phase 9 — Rendering Lists

React commonly uses `map()`.

## Example

```jsx
const users = ["Mohit", "John", "Alex"];

function App() {
  return (
    <ul>
      {users.map((user, index) => (
        <li key={index}>{user}</li>
      ))}
    </ul>
  );
}
```

# Phase 10 — Conditional Rendering

## Example


# Phase 11 — Forms

## Controlled Input

```jsx
import { useState } from "react";

function Form() {
  const [name, setName] = useState("");

  return (
    <div>
      <input
        value={name}
        onChange={(e) => setName(e.target.value)}
      />

      <h1>{name}</h1>
    </div>
  );
}
```

# Phase 12 — `useEffect`

Used for:

* API calls
* Side effects
* Timers

## Example

```jsx
import { useEffect } from "react";

function App() {
  useEffect(() => {
    console.log("Component Mounted");
  }, []);

  return <h1>Hello</h1>;
}
```

---

# Phase 13 — API Calls

## Example

```jsx
import { useEffect, useState } from "react";

function Users() {
  const [users, setUsers] = useState([]);

  useEffect(() => {
    fetch("https://jsonplaceholder.typicode.com/users")
      .then((res) => res.json())
      .then((data) => setUsers(data));
  }, []);

  return (
    <div>
      {users.map((user) => (
        <h3 key={user.id}>{user.name}</h3>
      ))}
    </div>
  );
}
```

# Phase 14 — Component Communication

## Parent → Child

Using props.

## Child → Parent

Using callback functions.

Example:

```jsx
function Child({ sendData }) {
  return (
    <button onClick={() => sendData("Hello Parent")}>
      Send
    </button>
  );
}
```

Install:

## Example


# Phase 16 — Custom Hooks

Reusable logic.

## Example

```jsx
import { useState } from "react";

function useCounter() {
  const [count, setCount] = useState(0);

  const increment = () => setCount(count + 1);

  return { count, increment };
}
```

# Phase 17 — Context API

* Mutating state directly
* Using index as key everywhere
* Too much prop drilling
* Large components
* Calling APIs incorrectly

For every topic:

1. Learn concept
2. Build tiny example
3. Modify example yourself
4. Break it intentionally
5. Fix it
6. Build mini project

That’s how React actually becomes intuitive.
