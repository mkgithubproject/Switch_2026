# 🧠 First: What Are Design Patterns?

👉 Think:

> **Design Patterns = Ready-made solutions to common coding problems**

Just like:

* 🏗️ Building house → you don’t reinvent structure every time
* 💻 Coding → you don’t reinvent architecture every time

# ⚡ Big Picture (Must Remember)

### 🧩 3 Types of Design Patterns

| Type           | Purpose              | Memory Trick      |
| -------------- | -------------------- | ----------------- |
| 🏗️ Creational | Object creation      | “How to create?”  |
| 🧱 Structural  | Object relationships | “How to connect?” |
| 🔁 Behavioral  | Object communication | “How to talk?”    |
